The entire website copy was rewritten in June 2026 based on BRAND.md (the Brand DNA document in the project root).

**Core brand direction:** The Practice is not a yoga company — it's a universe for reconnection (FORBINDELSE). The body is a compass. Rituals matter more than transformations. Rørvig is the physical heart of the brand.

**Rule:** Never define The Practice by what it is NOT. Only communicate what it IS.

**Key changes made:**
- "Privat" nav item renamed to "Brandsamarbejder" (links to private.html — URL unchanged)
- "Events" nav item renamed to "Oplevelser" (links to events.html — URL unchanged)
- Footer tagline added to all pages: "Et nordisk univers for forbindelse, ro og nærvær"
- private.html page purpose shifted from individual sessions to brand partnerships
- All copy across index, about, retreats, private, events, contact rewritten per brand DNA

**Why:** Kia provided a detailed BRAND.md document and a full copy brief covering all pages.

**How to apply:** When writing any new copy or suggesting changes, consult BRAND.md and follow these rules: calm/Nordic/warm/honest tone, never use "not" framing, center on FORBINDELSE.

**sensommer-retreat.html is excluded from all changes** — never touch it.
